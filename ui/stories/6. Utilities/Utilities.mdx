# CSS utility classes

## Margin and padding
- Applies `u-m-block`|`u-m-inline` and `u-p-block`|`u-p-inline` override classes.
- Uses CSS `clamp()` to set `min` and `max` values for larger gutters.

<Canvas of={Utilities.MarginPadding} />
<Controls of={Utilities.MarginPadding} />

## Screen readers
- `sr-only` visually hides content, but still make it available to screen readers.

<Canvas of={UtilitiesNoArgs.ScreenReader} />

## Hide content
- `u-hidden` removes content from the DOM, using `display: none;`.
- Alternatively, use the HTML `hidden` attribute.

<Canvas of={UtilitiesNoArgs.HideContent} />

## Center content
- `u-center` horizontally centers the content.

<Canvas of={UtilitiesNoArgs.CenterContent} />

## Make clickable
- `u-clickable` uses CSS `focus-within` to add a focus indicator to a component or block of content.
- Adding `data-module="make-clickable"` attribute to the HTML makes the entire component clickable via JavaScript.

<Canvas of={UtilitiesNoArgs.MakeClickable} />
