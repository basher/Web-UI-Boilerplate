# `<webui-popover>`
- Uses the native browser [popover API](https://developer.mozilla.org/en-US/docs/Web/API/Popover_API), along with [CSS anchor positioning](https://developer.mozilla.org/en-US/docs/Web/CSS/Guides/Anchor_positioning).
- Fallback behaviour uses the [`<webui-disclosure>`](/docs/web-components-or-custom-elements-webui-disclosure--docs) web component.

## Accessibility considerations
- The native browser `popover` content can be hidden via clicking the <kbd>esc</kbd> key or clicking outside the Web Component.

## Required and optional HTML  or `data-` attributes

Attribute | Behaviour
--- | ---
`popovertarget` | Required. Button that shows/hides the popover. Value should match the `id` of the popover content.
`popover` | Required. Popover content.
`id` | Required. Popover content. Value should match the `popovertarget` attribute of the button.
