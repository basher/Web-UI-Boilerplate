# `<webui-predictive-search>`
- Predictive search (with dynamic results from an API).
- This is a progressive enhancement of the [`<datalist>`](/docs/components-search--docs#predictive-search-using-datalist) search component, rather than using an [ARIA combobox](https://www.w3.org/WAI/ARIA/apg/patterns/combobox/).

<Canvas of={WebUIPredictiveSearch.WebUIPredictiveSearch} />

## Required and optional HTML  or `data-` attributes

Attribute | Behaviour
--- | ---
`data-fetch-url` | Required. On the web component. API endpoint for data to be fetched.
`data-fetch-target` | Required. On the search input that triggers the Fetch request. The value must match `data-fetch-container`.
`data-fetch-container` | Required. On the datalist. The value must match `data-fetch-target`.
