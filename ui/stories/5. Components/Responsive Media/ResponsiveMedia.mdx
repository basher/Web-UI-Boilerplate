# Responsive media (iframes, videos, etc)
- The `iframe` & `video` examples use standard Google Maps & YouTube `embed` code.
- Responsive media components can be nested inside other components - e.g. `card` or `grid`.
- Remove all `height` and `width` attributes from the embed code in order to use the default aspect ratio that has been defined in the CSS. This can be overridden by adding an inline style to the parent `responsive-media` container:
```
style="--mediaAspectRatio: 21/9;"
```

## Accessibility considerations
- `<iframe>` should have a non-empty `title` attribute.
