# Make clickable
- Utilises CSS `focus-within` to add a focus indicator to a component (e.g. card) when a child element receives keyboard `focus`.
- The entire component can be made clickable via JavaScript by adding a `data-module="make-clickable"` attribute to the component's HTML.
