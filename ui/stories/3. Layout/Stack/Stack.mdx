# Stack layout
- The stack layout injects a `gutter` (i.e. `margin-block-start`) between elements via their common parent.
- See [Every Layout's stack](https://every-layout.dev/layouts/stack/), [CubeCSS composition](https://cube.fyi/composition.html#examples) or [favourite 3 lines of CSS](https://andy-bell.co.uk/my-favourite-3-lines-of-css/) for a detailed explanation.
