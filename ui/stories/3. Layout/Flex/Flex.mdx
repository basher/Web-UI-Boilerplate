# Flexbox
- Flexbox should **not** be used for a grid system.
- Use something like [grid layout](/story/layout-grid--grid) or [asymmetric 2-col grid](/story/layout-grid--grid-asymmetric) for that.

- Please see [useful layout tips](/docs/layout-useful-layout-tips--docs).

## Override flex behaviour

Behaviour | Data attribute modifier
--- | ---
Align items | `data-align`
Change flex direction | `data-direction`
Change flex gaps | `data-flex-gap`
Make ALL flex items grow to fill up available space | `data-grow`
Justify content | `data-justify`
