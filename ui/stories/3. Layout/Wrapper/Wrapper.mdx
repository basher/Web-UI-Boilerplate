# Wrapper
- Simple wrapper layout component whose only job is to constrain width of content and apply auto margins.
- Normally, this would be used to divide content areas into _"slices"_, going down the page.
- For more complex layouts (e.g. multi-column), use one of the [grid layouts](/docs/layout-grid--docs).
- Ideally, there should be **NO visual styles** applied to wrappers (e.g. background colour).

- Use [stack layout](/story/layout-stack--stack) to apply vertical `gutters` between sibling wrappers.

- Use [utility classes](/story/utilities-helpers-and-utilities--margin-padding).
